# Hello world (echo, print)

## 1. Giới thiệu
Cách in ra màn hình trong PHP: `echo` và `print`. `echo` có thể nhận nhiều tham số, `print` trả về 1.

## 2. Ví dụ minh họa

```php
<?php
// Sử dụng echo
echo "Hello World!\n";
// Sử dụng print
print "Xin chào thế giới!\n";
?>
```

## 3. Bài tập
- 1. Tạo file PHP in ra tên bạn và tuổi bạn.
