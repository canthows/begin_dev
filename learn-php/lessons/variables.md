# Biến (Variables)

Biến trong PHP bắt đầu bằng ký hiệu `$` và không cần khai báo kiểu dữ liệu trước.

```php
<?php
// Khai báo biến
$name = 'Nguyen';
$age = 25;

// Sử dụng biến trong chuỗi
echo "Tên: $name, Tuổi: $age\n";
?>
```

**Giải thích:** PHP là ngôn ngữ loosely typed — biến tự động chuyển kiểu khi cần.
