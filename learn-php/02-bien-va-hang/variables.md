# Biến (Variables)

## 1. Giới thiệu
Biến trong PHP bắt đầu bằng $ và không cần khai báo kiểu. PHP tự động ép kiểu khi cần.

## 2. Ví dụ minh họa

```php
<?php
// Khai báo biến
$name = 'An';
$age = 20; // Lưu ý: phải có $ trước tên biến
// In ra chuỗi có biến
echo "Tên: $name, Tuổi: $age\n";
?>
```

## 3. Bài tập
- 1. Khai báo 3 biến: $a, $b, $c; in tổng của chúng.
